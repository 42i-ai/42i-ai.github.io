Portfólio de Projetos

1. Motiva-Ação (Flutter)

Um projeto desenvolvido em Flutter, focado em proporcionar uma plataforma para motivação e ação. Esse projeto inclui a integração de diferentes plataformas móveis para entregar uma experiência de usuário coesa em Android e iOS.

Repositório: Motiva-Ação

2. Jogo de Cartas em Unity

Um jogo de cartas desenvolvido em Unity, com ênfase em herança de classes em C#. O projeto inclui conceitos de design de jogos, como cartas de criatura e feitiço, além de regras de combate e gerenciamento de deck.

Repositório: Jogo de Cartas em Unity

3. Projeto Unity para UNINTER

Um projeto desenvolvido para fins acadêmicos, utilizando Unity para criar ambientes e interações 3D. Ideal para explorar mecânicas de jogos e cenários complexos em uma plataforma educacional.

Repositório: Unity Project - UNINTER
